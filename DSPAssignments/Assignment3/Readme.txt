Usage:
To run the script for this assignment, navigate to this folder in the terminal and execute the assignment by typing:

    chmod 755 Assignment3.swift
    ./Assignment3.swift
    
The CSV file can be imported into Microsoft Excel, iWork Numbers or OpenOffice Calc. Graphs have been plotted using Microsoft Excel for the report.

Sample files have been provided in case you do not want to set Swift up.

Dependencies:
macOS or Linux, Swift 3.1 or higher.
