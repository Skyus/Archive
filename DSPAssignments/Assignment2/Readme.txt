Usage:
To run the script for this assignment, navigate to this folder in the terminal and execute assignment1.sh by typing:

    ./Assignment2.swift > Data.csv

Note that this may take some time as a result of the Discrete Fourier Transform's complexity.

Data.csv can be imported into Microsoft Excel, iWork Numbers or OpenOffice Calc. Graphs have been plotted using Microsoft Excel for the report.

Dependencies:
macOS or Linux, Swift 3.0 or higher.
